# Personal Links Page

A dynamic personal links page builder, built with Next.js and deployed on Cloudflare Workers.

## Tech Stack

- [Next.js](https://nextjs.org/) – framework
- [TypeScript](https://www.typescriptlang.org/) – language
- [Tailwind](https://tailwindcss.com/) – CSS
- [better-auth](https://www.better-auth.com/) – auth
- [Resend](https://resend.com/) – emails
- [Cloudflare Workers](https://workers.cloudflare.com/) – deployments
- [Prisma](https://www.prisma.io/) – ORM (SQLite / D1)
